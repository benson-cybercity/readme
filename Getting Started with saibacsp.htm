    This documentation will help you get started with saibacsp.

    Saiba CSP (Content Service Provider)

    Siaba creates information, educational or entertainment content for e-delivery using gateways such as sms
    Saiba provides lee way for individuals or companies which want to deliver bulk sms or subscription services to a wide scope of content consumers (mobile phone subscribers)
    This is done through various telco's and to all consumer telco's

    Saiba provides our customers with two products which a user is at liberty to choose. This products are
    1. Dashboard
    This is a graphical user interface where one can schedule, send bulk sms, see subscriptions to their short code and view and download reports
    2. API Interface
    This is an application programming interface where users can write their application on top of our application layer. This allows one to send sms, schedule sms, get remaining sms count , and delivery reports, also allows one to get notifications for subscriptions and unsubscriptions
